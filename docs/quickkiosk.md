# Quick Kiosk

The quick kiosk is a self-check-in screen used by members to register for classes and mark attendance even more quickly than the <a href="/docs/kiosk">normal kiosk</a>.  It is always located at yourdomain.managememberships.com/qkiosk 

---

## Process

The process is nearly identical to the <a href="/docs/kiosk">normal kiosk</a>, HOWEVER, there is no manual selection of classes and the input box is automatically focused.  This allows you to set up a QR code scanner and use the wallet codes to check in quickly.
