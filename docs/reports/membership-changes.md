# Membership Changes

The **Membership Changes** report tracks new signups, cancellations, and other membership status changes over time. It helps you understand growth trends and identify patterns in member churn.

---

## Filters

### Date Range
Select a **Start Date** and **End Date** to define the reporting period.

### Quick Date Ranges
- **Current Month** - From the first of the current month to today
- **Last Month** - The full previous calendar month
- **Last 3 Months** - From 3 months ago to today

### View Mode
- **Monthly** - Aggregate changes by month
- **Daily** - View individual daily changes

### Location Filter
If multi-location is enabled, filter by a specific location or view all locations combined.

---

## Report Data

The report shows membership activity including new member signups, cancellations, and net changes for the selected period, helping you visualize growth or decline trends.
