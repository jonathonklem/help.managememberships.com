# Time Clock Audits

The **Time Clock Audits** report provides a history of all timesheet-related activity including clock-ins, shift changes, time-off requests, and staff availability updates.

---

## Filters

### Search
Search audit entries by staff member name or details.

### Event
Filter by the type of audit event (e.g., created, updated, deleted).

### What
Filter by the type of timesheet record:
- **Clock Ins** - Staff clock-in and clock-out records
- **Shifts** - Scheduled shift entries
- **Request Offs** - Time-off requests
- **Staff Availability** - Weekly availability changes

### Date Range
Select a start and end date/time to narrow the audit window. Defaults to the last 7 days.

---

## Audit Details

Each audit entry shows the change history including what was modified, the old values, new values, who made the change, and when it occurred.
