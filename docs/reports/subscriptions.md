# Subscriptions

This report helps you see which members are subscribed to a specific **membership level**, including both their **main subscription** and any **add-ons** they’ve purchased.

## 🔍 How to Use

1. Use the dropdown at the top to select a **membership level**.
2. The table will automatically refresh with a list of members who have that level.
3. For each user, you'll see:
   - **User name**
   - **Amount** (price per unit in dollars)
   - **Quantity** (how many units they’re subscribed to)
   - **Total** (amount × quantity)

## 💰 At the Bottom

- A **Total** row shows the total revenue for that selected membership level, including all users who have it either as a **primary subscription** or an **add-on**.

## Example Use Case

If you offer something like "Unlimited Classes" or "Family Plan + Kids Add-On", this report helps you:
- See which users are on that level
- Understand how many are using it as an add-on vs primary subscription
- Track overall revenue tied to that level

## 📝 Notes

- This report includes both:
  - Members who have the selected level as their **main subscription**
  - Members who have it as an **add-on**
- Everything updates in real-time when you select a different membership level.
