# Financial


You can view all incoming payments, fees, and net totals for your gym using the **Financial Transactions** report.

## 📆 How to Use

1. Select a **start date** and an **end date** using the date pickers at the top.
2. Once you choose the end date, the report will automatically load transactions between the two dates.
3. The table will show:
   - **Date** of the transaction
   - **Member name**
   - **Description** of the transaction
   - **Amount In** (what was charged)
   - **Fee** (processing or platform fees)
   - **Net Amount** (your gym’s final amount)

## 🧮 Totals

At the bottom of the table, you'll see totals for:
- **Amount In**
- **Fee**
- **Net Amount**

## 📝 Notes

- Transactions made by the **owner account** are automatically excluded.
- If your selected start date is after the end date, the report will fix it for you.
- The table updates automatically — no need to click any "submit" button.
