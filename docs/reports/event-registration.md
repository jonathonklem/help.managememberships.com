# Event Registration

The **Event Registration** report allows you to view registrations for your events, see who has signed up, and manually add attendees.

---

## How to Use

1. Select an **Event** from the dropdown to view its registrations.
2. The report shows all registered attendees with their details.

---

## Manual Registration

You can manually register members for events from this report:
1. Search for a member by name
2. Select a price/ticket option
3. Set the number of tickets
4. Optionally mark as **comp** (complimentary, no charge)
5. Answer any required event questions

---

## Location Filter

If multi-location is enabled, you can filter events by location.
