# Birthday Report

The **Birthday Report** helps you find members with upcoming birthdays so you can send personalized messages, plan celebrations, or run birthday promotions.

---

## Filters

### Date Range
Select a **Start Date** and **End Date** to find members whose birthdays fall within that window. Defaults to the current calendar month.

### Membership Level
Optionally filter by one or more membership levels to narrow results.

---

## Results

The report lists members whose birthdays fall within the selected date range, showing:
- Member name and contact information
- Birthday date
- Membership level

---

## Actions

You can select members from the results and initiate a text or email campaign directly from this report, making it easy to send birthday greetings or promotional offers.
