# Resource Report

The **Resource Report** shows bookings and usage data for your resources (rooms, equipment, facilities).

---

## Report Columns

The report displays:
- **Start Time** - When the booking begins
- **End Time** - When the booking ends
- **Member** - Who made the booking
- **Bill COF** - Whether the member's card on file was charged
- **Resource** - Which resource was booked

Use this report to track resource utilization, monitor bookings, and plan capacity.
