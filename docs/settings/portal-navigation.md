# Portal Navigation

The **Portal Navigation** section allows you to customize the navigation menu on your member-facing portal. You can add links to external pages, internal resources, or organize items into dropdown menus.

---

## Adding a Navigation Item

Click **Add Navigation Item** to create a new menu link. The following fields are available:

### Label
The text displayed in the navigation bar (e.g., "Schedule", "Contact Us").

### URL
The web address the link points to. Can be a full URL for external sites or a relative path for internal pages.

### Type
Choose the navigation item type:
- **Link** - A standard clickable link
- **Dropdown** - A parent menu item that contains child links

### Parent Dropdown (Optional)
If this item should appear inside an existing dropdown menu, select the parent item here.

### Open in New Tab
When enabled, clicking this link opens the destination in a new browser tab.

### Require Login
When enabled, this navigation item is only visible to logged-in members. Guests and visitors will not see it.

---

## Current Navigation Items

Your existing navigation items are listed in order. You can edit or remove items as needed.

---

## Tips

- Use **dropdowns** to group related links and keep your navigation clean.
- Use **Require Login** for member-only pages like schedules or account resources.
- Keep labels short and descriptive for the best display on mobile devices.
