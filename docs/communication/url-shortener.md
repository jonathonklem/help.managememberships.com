# URL Shortener

The URL Shortener creates compact, trackable links that can be used in text campaigns, emails, banners, and other communications. Short URLs use the `mbrs.app` domain for a clean, professional appearance.

---

## Overview

Short URLs are automatically created when you use features like referral links or `{refer_link}` variables in campaigns. They provide click tracking and analytics for your links.

## URL Details

Each short URL tracks the following information:

| Field | Description |
|-------|-------------|
| **Short URL** | The shortened link (e.g., `mbrs.app/abc123`) |
| **Destination** | The original full URL the short link redirects to |
| **Clicks** | Total number of times the link has been clicked |
| **Last Click** | When the link was last clicked |
| **Created** | When the short URL was created |
| **QR** | A QR code that encodes the short URL for easy scanning |
