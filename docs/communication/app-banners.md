# App/Portal Banners

App/Portal Banners allow you to display announcement messages across the top of your member portal and mobile app. Use banners for promotions, important notices, schedule changes, or any information you want members to see prominently.

---

## Creating a Banner

Click **"Create Banner"** to open the banner form.

### Banner Fields

#### **Banner Message**
The text displayed in the banner (max 255 characters). You can use `{refer_link}` to insert the member's referral link, which renders as a clickable link on the web portal.

#### **Start Date**
When the banner should begin displaying. Defaults to the current date and time.

#### **End Date**
When the banner should stop displaying. Leave empty for an open-ended banner that displays indefinitely.

#### **Active**
Toggle the banner on or off. Defaults to active.

#### **Priority**
A numeric value to control the display order when multiple banners are active. Lower numbers display first.

---

## Managing Banners

The banner table shows all your banners with their:

- **Active** status (icon)
- **Message** content
- **Start Date**
- **End Date**
- **Priority**

You can edit or delete banners from the table actions.
