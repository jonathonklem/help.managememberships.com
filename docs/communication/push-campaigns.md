# Push Campaigns

Push Campaigns allow you to send push notifications directly to members who have the mobile app installed. These notifications appear on members' phones even when they're not actively using the app.

---

## Creating a Push Campaign

Click **"Create New"** at the top to open the campaign form.

### Campaign Fields

#### **Subject**
The title/heading of the push notification. Keep it concise and attention-grabbing (max 255 characters).

#### **Schedule Send Time**
Pick the date and time you want the notification sent. Defaults to the current time.

> **Tip:** To save a campaign as a draft, set the scheduled time far into the future.

#### **Message**
The body of the push notification. This is the main content members will see.

#### **Membership Level**
Filter the member list by one or more membership levels. This narrows down which members appear in the selection list.

#### **Selected Members**
Choose which members will receive the notification. Only members with the mobile app installed will appear as options. You can click **Select All** to include all eligible members.

### Send Test
Before scheduling, you can send a test push notification to a specific member to preview how it will look on their device.

---

## Viewing Campaigns

The campaign table shows:

- **Subject** - The notification title
- **Schedule Send Time** - When the notification is set to go out
- **Completed Send Time** - When the notification was actually delivered

You can delete campaigns from the table as needed.
